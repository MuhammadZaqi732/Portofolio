Zakyy.m Welcome - Vite + Tailwind scaffold
---------------------------------------
How to run:

1. Install dependencies:
   npm install
2. Run dev server:
   npm run dev
3. Build:
   npm run build
4. Preview build:
   npm run preview

This scaffold includes:
- src/index.html (Landing)
- src/about.html (About)
- src/main.js
- src/style.css (Tailwind imports + custom)
- public/assets/profile.jpg (placeholder)
